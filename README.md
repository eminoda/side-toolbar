# 侧边工具栏 side-toolbar

> 我希望认真落地一款能帮助到每个人的小工具软件

## 希望它能解决的问题

- 自带定制的浏览器
  - 提供黑暗主题：护眼
  - 分类 tab 栏：让标签页更好管理
    - TODO(优化)：采用 qiankun entry-html 来加速加载 tab 标题
- 开发工具
  - 页面 xhr 接口生成 Mock 文档
  - 页面 302 拦截
  - 对 cookies，sessionstorage 等变化进行提示
  - 抓包
- 日常工具
  - 和手机进行交互：文件传递，文本复制
  - 截图，取色
  - pdf 文件生成
  - ...
- 广告
